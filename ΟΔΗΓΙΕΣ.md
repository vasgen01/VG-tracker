# VG Tracker — Οδηγός Εγκατάστασης

## Βήμα 1: Δημιούργησε GitHub account
1. Πήγαινε στο **github.com**
2. Κάνε κλικ **Sign up**
3. Email, password, username (π.χ. `vasilis-genidounias`)
4. Επιβεβαίωσε το email σου

## Βήμα 2: Δημιούργησε νέο repository
1. Μετά το login, κάνε κλικ **+** (πάνω δεξιά) → **New repository**
2. Repository name: `vg-tracker`
3. Επέλεξε **Public**
4. Κάνε κλικ **Create repository**

## Βήμα 3: Ανέβασε τα αρχεία
1. Στο repository που μόλις έφτιαξες, κάνε κλικ **uploading an existing file**
2. Σύρε και άφες (drag & drop) ΟΛΑ τα αρχεία από τον φάκελο `vg-tracker-app`:
   - index.html
   - manifest.json
   - sw.js
   - icons/ (φάκελος με icon-192.png και icon-512.png)
3. Κάνε κλικ **Commit changes**

## Βήμα 4: Deploy στο Netlify (δωρεάν)
1. Πήγαινε στο **netlify.com**
2. Κάνε κλικ **Sign up** → επέλεξε **GitHub** για login
3. Κάνε κλικ **Add new site** → **Import an existing project**
4. Επέλεξε **GitHub** → επέλεξε το repository `vg-tracker`
5. Άφησε όλες τις ρυθμίσεις ως έχουν → **Deploy site**
6. Σε 1 λεπτό θα σου δώσει URL: π.χ. `https://amazing-name-123.netlify.app`
7. (Προαιρετικό) Άλλαξε το URL: **Site settings** → **Change site name** → `vg-tracker-vasilis`

## Βήμα 5: Εγκατάσταση στο iPhone
1. Άνοιξε το URL στο **Safari** (όχι Chrome — μόνο Safari υποστηρίζει PWA στο iOS)
2. Πάτα το κουμπί **Share** (το τετράγωνο με το βελάκι)
3. Σκρολ κάτω και πάτα **"Add to Home Screen"**
4. Όνομα: `VG Tracker` → **Add**
5. Τώρα έχεις app icon στην αρχική οθόνη 🎉

## Βήμα 6: Εγκατάσταση στο PC (Chrome)
1. Άνοιξε το URL στο **Chrome**
2. Στη γραμμή διευθύνσεων θα εμφανιστεί εικονίδιο εγκατάστασης (⊕)
3. Κάνε κλικ → **Install**
4. Εμφανίζεται σαν κανονική desktop εφαρμογή

## Σημαντικό — Ενημερώσεις
Κάθε φορά που θέλεις να κάνεις αλλαγές:
1. Ανέβασε το νέο `index.html` στο GitHub (αντικατάσταση)
2. Το Netlify ανανεώνεται αυτόματα σε ~30 δευτερόλεπτα
3. Στο κινητό: ανανέωσε τη σελίδα μία φορά για να πάρει τις αλλαγές

## Τα δεδομένα σου
Αποθηκεύονται τοπικά στο κινητό/PC σου (localStorage).
Δεν χάνονται αν κλείσεις την εφαρμογή.
Αν αλλάξεις συσκευή, τα δεδομένα δεν μεταφέρονται αυτόματα.
